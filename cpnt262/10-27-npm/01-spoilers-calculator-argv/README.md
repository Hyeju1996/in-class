# CPNT 262 Assignment 4
A simple command line calculator.

## Usage
Enter a command in the following format:

```
$ node app <num1> <operation> <num2>
```

## Examples
1. `$ node app 4 plus 5`
2. `$ node app 8 minus 3`
3. `$ node app 6 times 5`
4. `$ node app 8 div 0`
